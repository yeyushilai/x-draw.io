# x-draw.io 图表仓库

## 项目简介
`x-draw.io` 是一个专门用于存放通过 [draw.io](https://app.diagrams.net/) 网站绘制的各类图表的仓库。

## draw.io 工具介绍
`draw.io` 是一款功能强大的免费在线图表绘制工具，支持流程图、网络 diagram、UML 图、ER 图、数据库 schema 设计、BPMN 图、电路图等多种图表类型。

## 如何使用

1. 在 [draw.io](https://app.diagrams.net/) 网站上创建和编辑图表
2. 完成图表绘制后，选择导出功能
3. 将导出的图表文件上传到本仓库的相应目录中

## 目录结构

- **common/**: 通用图表
- **custom/**: 自定义图表
- **drafts/**: 草稿图表
- **projects/**: 项目相关图表

## 如何提交 Issue

如果您遇到问题、有建议或需求，可以通过以下步骤提交 Issue：

1. 访问项目的 GitHub 仓库页面
2. 点击页面顶部的 "Issues" 选项卡
3. 点击绿色的 "New issue" 按钮
4. 选择 "图表相关 issue" 模板
5. 填写 Issue 标题和详细内容
6. 添加适当的标签（如有需要）
7. 点击 "Submit new issue" 按钮提交
